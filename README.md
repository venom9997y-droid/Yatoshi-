# Yatoshi — Python Terminal for Android

A powerful Python terminal for Android. Open `.py` files directly from your file manager!

## Features
- Full Python 3 terminal
- Run `.py` files directly from file manager
- `pip install` any package
- Command history
- Kill running processes
- Looks and feels like Termux

## How to build

This APK is built automatically via GitHub Actions.

1. Push this repo to GitHub
2. Go to **Actions** tab
3. Click **Build Yatoshi APK** → **Run workflow**
4. Wait ~40 minutes
5. Download APK from **Artifacts**

## Usage

- Open Yatoshi app → type commands in the terminal
- Or open any `.py` file in your file manager → choose Yatoshi → runs instantly

## Commands

```
python <file.py>     — run a script
pip install <pkg>    — install package
pip list             — list packages
ls / cd / pwd        — navigate files
clear                — clear screen
help                 — show help
```
